Thunderbird
===========

Ansible role for installing Thunderbird.

Requirements
------------

- Ansible 2.14 or later.

Role variables
--------------

Dependencies
------------

No dependencies.

Example Playbook
----------------

```
---
- hosts: localhost
  tasks:
    - ansible.builtin.include_role: thunderbird
```
