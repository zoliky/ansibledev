Teamviewer
==========

Ansible role for installing Teamviewer.

Requirements
------------

- Ansible 2.14 or later.

Role variables
--------------

Dependencies
------------

No dependencies.

Example Playbook
----------------

```
---
- hosts: localhost  
  tasks:
    - ansible.builtin.include_role: teamviewer
```
